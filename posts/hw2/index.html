This code is a web crawling and scraping framework in Python. It uses a spider to scrape. The spider navigates through movie pages on themoviedb.org, extracts information about actors, and retrieves details about their movies or TV shows.

Step 4: First parsing method

# assuming we are starting on the movie page, and then navigating to the Full Cast & Crew Page
def parse(self, response):
    """
    Parses TMDB movie pages.
    Assumes that you start on a movie page, and then navigates to the Full Cast & Crew page.
    Yields a scrapy.Request for the "Full Cast & Crew" page, with the parse_full_credit method specified in the callback argument.
    Does not return any data.
    """
    # full cast and crew page has url: <movie_url>cast
    full_credits_url = response.url + "/cast/"
      
    # yielding a scrapy request using full_credits_url and callback
    # callback means calling parse_full_credits(self, responce) using the full_credits_url ( so we have navigated -
    # to the full cast and crew page
    yield scrapy.Request(full_credits_url, callback=self.parse_full_credits)

In the parse method, the spider navigates from the movie page to the Full Cast & Crew page by adding “/cast/” to the current URL. A scrapy request is yielded with the URL of the Full Cast & Crew page and the callback function parse_full_credits.

Step 5: Second Parsing Method

    # parse_full_credits to get the actor URLs from the Full Cast & Crew page
def parse_full_credits(self, response):
     """
     Parses TMDB "Full Cast & Crew" pages.
     Yields a scrapy.Request for the page of each cast member, with the parse_actor_page method specified in the callback argument.
     Does not return any data.
     """
   
    # this gets each actor URL from the full cast and crew page using CSS selector   
    #gets full info for all actors
     actors = response.css('div.info')
        
     # extracts indidual urls of each actor
     actors = actors.css('div.info > p > a::attr(href)').getall()
        

       # iterates thru each actor URL and yields a request for the actor url and the parse_actor_page as the callback
     for urlx in actors:
        #yield {"actor" :"https://www.themoviedb.org" + urlx}
        yield scrapy.Request("https://www.themoviedb.org" + urlx, callback=self.parse_actor_page)

The parse_full_credits method extracts actor URLs from the Full Cast & Crew page using CSS selectors. For each actor URL, a scrapy request is yielded with the actor URL and the callback function parse_actor_page.

Step 6: Final Parsing Method

# parse_actor_page to get actor name and their movies/tv shows from the actor's page
def parse_actor_page(self, response):
    """
    Parses TMDB actor pages.
    Yields a dictionary containing the actor's name keyed by "actor" and the movie/TV show keyed by "movie_or_TV_name" for each movie/TV show they have acted in.
    """
     
    # get the actor's name from the actor's page
    name = response.css('div.title > h2 > a::text').get()


        
    # looks for acting header, saves it to index
    index = response.css('div.credits_list h3::text').getall().index("Acting")
        
    # looks for acting roles (exluding all other roles), so it uses the index of the acting header
    roles = response.css('div.credits_list table.card.credits')[index]
        
    # gets name of the roles
    final = roles.css('a.tooltip bdi::text').getall()
        
    # iterates through each move/tv show and gets a dictionary with actor and movie/TV show names
    for movie_or_TV_name in final:
            yield {"actor": name, "movie_or_TV_name": movie_or_TV_name}

The parse_actor_page method extracts the actor’s name from the actor’s page. It finds the index of the “Acting” header and extracts the roles (movies/TV shows). It only takes the movies/TV shows the actor starred in, and leaves out any set they worked on as a crew or production.

A dictionary with actor name and movie/TV show details is yielded for each entry. This dictionary displays all the movies/tv shows each actor in the movie additionally worked on

The spider, when run with the command scrapy crawl tmdb_spider -o movies.csv -a subdir=671-harry-potter-and-the-philosopher-s-stone, will start from the specified movie page on themoviedb.org, navigate to the Full Cast & Crew page, and collect information about actors and their associated movies or TV shows. The output will be saved to a CSV file named movies.csv.
